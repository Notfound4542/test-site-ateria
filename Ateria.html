Comment jouer ?

Une fois que vous avez terminé de télécharger Ateria. Veuiller suivre les étapes ci dessous:

  1. Clique droit sur le fichier puis "extraire ici"(extraire le fichier en .zip sur votre bureau)
  2. Ouvrir le fichier Ateria Installer, qui va permettre d'installer tout les modules nécessaire pour le bon fonctionnement du jeux
  3. Une fois Ateria Installer lancé veuiller vous rendre dans le dossier "Ateria"
  4. Un fichier sera apparu sous le nom de "Ateria game", cliquer puis jouer
  5. Et voila le setup du jeux est terminé, vous pouvez maintenant jouer a ATERIA

Si vous avez eu un problème en particulier durant le setup du jeu, veuiller contacter notre équipe
